# Tags energéticos · Buscador Elemental

Aplicar **uno o más** de estos tags a cada producto en Shopify Admin → Productos → [editar] → **Tags**.

El buscador (`sections/duendes-home-v2-buscador-elemental.liquid`) normaliza la búsqueda (minúsculas, sin acentos) y cruza con el mapa `SINONIMOS` en JS; los tags en tienda deben coincidir con la forma **`esencia-*`** de la primera columna.

| Tag | Sinónimos que lo activan (vienen del mapa SINONIMOS del buscador) |
| --- | --- |
| `esencia-proteccion` | protección, proteger, escudo, defensa, cuidado, amparo |
| `esencia-calma` | calma, paz interior, tranquilidad, serenidad, relax, ansiedad |
| `esencia-amor` | amor, pareja, vínculo, corazón, romance, afecto |
| `esencia-fuerza` | fuerza, poder, coraje, valentía, empuje, energía |
| `esencia-abundancia` | abundancia, prosperidad, dinero, riqueza, éxito, trabajo |
| `esencia-sanacion` | sanación, sanar, curar, salud, recuperación |
| `esencia-paz` | paz, hogar, armonía, equilibrio |
| `esencia-sabiduria` | sabiduría, claridad, intuición, decisiones, guía |
| `esencia-hogar` | hogar, casa, familia, raíces |
| `esencia-transformacion` | transformación, cambio, renacer, nueva etapa, ciclo |

**Recomendación:** cada Guardián puede llevar **2–3** tags, no más. Si lleva todos, deja de ser específico.

**Opcional:** el mapa incluye `sueños → magia`; si querés que matchee, podés usar el tag adicional `esencia-magia` en Admin (fuera de la tabla principal).
